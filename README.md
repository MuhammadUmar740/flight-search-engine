# Flight Search Engine

This is a React-based flight search engine application that allows users to search for flights, sort them by price or airline, and visualize the results in a chart.

## Features

- Search for flights by specifying departure and return dates, origin, and destination.
- Sort flight results by price or airline in ascending or descending order.
- View flight details in a list format.
- Visualize flight prices by airline in a chart.

### Key Files

- **[components/SearchFlight.tsx](components/SearchFlight.tsx)**: Main component for the flight search functionality.
- **[components/CitySearch.tsx](components/CitySearch.tsx)**: Component for selecting origin and destination cities.
- **[components/FlightRecords.tsx](components/FlightRecords.tsx)**: Component for displaying the list of flight results.
- **[components/Chart.tsx](components/Chart.tsx)**: Component for visualizing flight prices by airline.
- **[app/api/flights-search.ts](app/api/flights-search.ts)**: API function to fetch flight data.

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd flight-search-engine
   ```
2. Install dependencies:
    ```bash
    npm install
    ```

Create a .env.local file and configure any necessary environment variables.

## Running the Application
  To start the development server, run:
  ```bash
    npm run dev
  ```

The application will be available at http://localhost:3000.

## Usage

1. Enter the origin and destination cities using the city search inputs.
2. Select departure and return dates.
3. Click the Search Flights button to fetch flight data.
4. Use the Sort By Price and Sort By Airline buttons to sort the results.
5. View the flight details in the list and the price visualization in the chart.
   
## Technologies Used

- React: Frontend library for building the user interface.
- Next.js: Framework for server-side rendering and routing.
- TypeScript: For type safety and better developer experience.
- Tailwind CSS: For styling the application.

## Folder Structure

- components/: Contains reusable React components.
- app/: Contains API-related files, logic and UI.
- public/: Static assets like images and icons.
- .next/: Build output directory (auto-generated by Next.js).
